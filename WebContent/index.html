This web application provides a guide to mountain biking in the Washington D.C. metro area. Mountain biking trails are represented as icons on the map. These icons can be clicked to view information about the trail including its name, address, distance, difficulty and condition. The user can also read reviews about the trail written by other users and you can submit your own review of the trail. Using the controls in the upper right hand corner of the map the user can search for a particular trail by name or style the trail map icons by various criteria including trail condition, average trail rating and trail difficulty.
